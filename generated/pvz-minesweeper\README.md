# pvz-minesweeper

A fully functional Minesweeper game with Plants vs. Zombies theme, featuring responsive design, game logic, and thematic UI elements.

## Type

static_web_prototype

## Run

- `Open index.html directly in a browser`
- `Optional: python -m http.server 8000`
